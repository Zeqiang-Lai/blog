    首先要导入包

    • 一般 pyplot 就能满足基本画图需求, plt 为通用的缩写
    • numpy 不是必须的，这里是为了生成数据

    Basic

    基本的二维曲线图

    • figure 指的是呈现图片窗体，后面的一系列代码都是作用在这个窗体上的
    • plot 有很多参数，具体可以看 matplotlib 的 API
    • 最后一定要 show() 把窗体显示出来

    Axis

    Axis 是指窗体里的框框，上下左右四条线

    下面展示一些自定义 Axis 的方法

    • xlim, ylim 控制轴的取值范围
    • xlabel, ylabel 控制轴的说明文字
    • xticks, yticks 可以自定义轴上显示的分点

    Latex支持： 使用$将latex表达式扩起来，记得用 r 忽略特殊字符 \ ，所以一般这么用 r’$ expreesion $‘

    改变坐标轴位置

    先获取当前坐标轴，然后利用 spines 这个变量获取各个轴，利用内置的函数改变其位置。

    标注

    标注比较麻烦，需要用到 plt.annotate() 这个函数

    • 第一个参数是要显示的标注
    • xy 是文字要标注的位置
    • xycoords 声明 xy 的计算方式
    • xytext 为文字显示的位置（相对于xy）
    • textcoords 声明xytext的计算方式
